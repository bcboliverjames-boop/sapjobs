隐私政策

最后更新日期：2024年12月14日

1. 信息收集

本网站（自由职业收入计算和工时导出）是一个纯前端应用，所有计算都在您的浏览器本地完成。我们不会收集、存储或传输您的任何个人信息或计算数据。

2. 本地存储

为了提升用户体验，本网站可能会在您的浏览器本地存储以下信息：

  • 主题偏好：您选择的深色/浅色模式设置
  • 节假日数据缓存：用于快速加载节假日信息的缓存数据

这些数据仅存储在您的设备上，不会发送到任何服务器。

3. 第三方服务

本网站使用以下第三方服务：

  • Google Analytics：用于分析网站访问情况，帮助我们了解用户如何使用本网站。Google Analytics 可能会收集您的 IP 地址、浏览器类型、访问时间等匿名信息。您可以访问 Google 隐私政策 了解更多信息。
  • GitHub：用于加载节假日数据。我们仅从公开的 GitHub 仓库读取数据，不会向 GitHub 发送任何个人信息。

4. Cookie 使用

本网站可能使用 Cookie 和类似技术来存储您的偏好设置。这些 Cookie 仅用于改善用户体验，不会用于追踪或识别您的身份。

5. 数据安全

由于所有计算都在您的浏览器本地完成，您的计算数据不会离开您的设备。我们不会存储、传输或访问您的任何计算数据。

6. 儿童隐私

本网站不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

7. 隐私政策变更

我们可能会不时更新本隐私政策。任何变更都会在本页面上发布，并更新"最后更新日期"。我们建议您定期查看本页面以了解最新信息。
