About Us

Project Background

The Freelancer Income Calculate and Timesheet Export tool is a practical tool developed by an SAP freelancer to help freelancers (especially SAP consultants) quickly and accurately calculate project income and export timesheet.

As freelancers, we often need to calculate income for different project periods, considering factors such as leave, overtime, and holidays. Traditional manual calculations are both tedious and error-prone, so we developed this tool to make income calculation simple and efficient.

Design Philosophy: Simple, accurate, and practical. All calculations are performed locally in your browser, protecting your privacy. No registration required, completely free.

Core Features

Multi-Country Support

Supports China, United States, and Europe, automatically identifies statutory holidays and currency

Flexible Salary Types

Supports daily, hourly, and monthly calculation methods to meet different project needs

Leave & Overtime Management

Supports multiple leave and overtime ranges for precise calculation of actual workdays

Smart Holiday Recognition

Automatically excludes weekends and statutory holidays, supports makeup workday recognition

Detailed Export

One-click Excel export with complete paid days and excluded dates details

Privacy Protection

All calculations performed locally, data not uploaded to server, protecting your privacy

Target Audience

This tool is particularly suitable for:

  • SAP Freelancers: Quickly calculate project income, manage leave and overtime
  • Other IT Freelancers: Calculate project income by day/hour/month
  • Freelance Designers, Copywriters, etc.: Need to accurately calculate workdays and income
  • Project Management Personnel: Need to calculate project costs and income

Technical Features

  • Pure Frontend: No backend server required, all calculations performed locally in browser
  • Responsive Design: Perfect adaptation for desktop, tablet, and mobile
  • Apple-style UI: Clean and beautiful user interface with excellent user experience
  • Dark Mode Support: Automatically adapts to system theme, supports manual switching
  • Multi-language Support: Available in Chinese and English
  • Data Caching: Intelligent holiday data caching to improve loading speed

Future Plans
